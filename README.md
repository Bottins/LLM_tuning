# SciML-LLM Fine-tuning

Fine-tuning of Qwen3 on a Scientific Machine Learning Q&A dataset covering PDEs, PINNs, GNNs, and optimization methods. Supports single-GPU (local, 4-bit) and multi-node distributed training (FSDP, 16 GPUs).

## Structure

```
sciml-llm-finetune/
├── dataset_builder.py     # generates train/val Q&A dataset
├── finetune.py            # LoRA + FSDP training script
├── accelerate_fsdp.yaml   # Accelerate config for multi-node FSDP
├── job.pbs                # CINECA PBS job script (2 nodes, 16 GPUs)
├── requirements.txt
└── data/                  # generated by dataset_builder.py
```

## Quickstart (local, 1 GPU)

```bash
pip install -r requirements.txt
python dataset_builder.py
python finetune.py --model_id Qwen/Qwen3-1.7B --local
```

## CINECA (16 GPU, 2 nodes)

```bash
python dataset_builder.py          # build dataset first
qsub job.pbs                       # submit job
qstat -u $USER                     # monitor
```
